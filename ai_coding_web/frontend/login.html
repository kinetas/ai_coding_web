동작 방식

  • 로그인 성공 시 브라우저에 Supabase 세션이 저장되고, 원래 가려던 페이지(next)로 이동합니다.
  • 프로젝트에서 이메일 확인을 켜 두면 회원가입 직후에는 메일 인증이 필요할 수 있습니다.
  • 내 분석 만들기/보기 페이지는 진입 시 로그인 여부를 확인하고, 미로그인이라면 이 페이지로 이동합니다.
